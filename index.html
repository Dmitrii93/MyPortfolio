Навыки

Что я использую в работе

HTML5

Отсюда я начинаю каждый проект. Здесь я создаю каркас сайта с SEO - оптимизированной структурой.

CSS3

С языком стилей я создаю абсолютно любой внешний вид страниц. Конечно же я использую препроцессоры в своей работе. (SASS/SCSS)

Java Script

Этот язык программирования позволяет оживить все что угодно. С ним я создаю слайдеры, окна, подсказки, вкладки, анимации и многое другое.

JQuery

Даже с последними стандартами JS библиотека Jquery все еще актуальна. Использую не всегда, но навык работы с ней присутствует.

React JS

Библиотека React позволяет создавать SPA. В настоящее время я работаю большую часть времени, используя React с последними новшествами.

React Native

Этой зимой я впервые, стал частью проекта, основанном на React Native. Это был приятный опыт, где я отвечал за клиентскую часть, где использовал REDUX и платформу EXPO.

Node Package Manager

Установка пакетов производится при помощи NPM.

Node.js

JS, прекрасно работающий с серверной частью приложения. Работаю с использованием фрейморка express.js.

Gulp

Рутинную работы я перекидываю на Gulp. Имею свою сборку. По необходимости расширяю ее другими возможностями, или пользуюсь уже специально созданной.

Bootstrap

К библиотеке Bootstrap прибегаю лишь по требованию, но навык работы с ней имеется, как и с другими библиотеками, вроде materialize, chakra и т.д.

Wordpress

Посадив сайт на движок WP, можно позволить заказчику изменять контент без необходимости изменения кода.

Git

С Git-ом сохраняется информация о всех изменениях в коде, начиная с самой первой строчки и обеспечивается удобная работа в команде над проектом в целом.
